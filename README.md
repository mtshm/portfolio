## アプリの概要
食事の栄養管理アプリです。
一日に必要なタンパク質の量を満たす食事のメニューを作成できます。

## 使用技術
Vue.js ( Vue-CLI, Vuex, Vue-router,  vuex-persistedstate, Vuetify), axios, Lodash, npm
